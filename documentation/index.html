Template Structure

Full list of plugins and libraries is available in Credits section of the documentation.

Stylesheets

  • css/vendors.css
    Bundle of libraries styles.
  • css/style.css
    Contains template styles with table of contents.

Scripts

  • js/vendors.js
    Bundle of libraries scripts.
  • js/main.js
    Contains template functionality with table of contents.

Images

  • img/
    Images of the template sections. There are grey placeholders which you should replace with your own images.

Fonts

  • Google Fonts
    All the fonts are served from Google Fonts server.
  • Font Awesome
    Font Awesome icons.

HTML Structure

The general template markup is the same throughout the files. Please make sure that you have the correct HTML markup first before doing any customizations.

          
            <!DOCTYPE html>
            <html lang="en" data-x="html" data-x-toggle="html-overflow-hidden">
            <head>
              <!-- Required meta tags -->
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1">

              <!-- Google fonts -->
              <link rel="preconnect" href="https://fonts.googleapis.com">
              <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
              <link href="https://fonts.googleapis.com/css2?family=Cormorant+Garamond:wght@400;500;600;700&family=Jost:wght@400;500;600;700&display=swap" rel="stylesheet">
                
              <!-- Stylesheets -->
              <link rel="stylesheet" href="css/vendors.css">
              <link rel="stylesheet" href="css/main.css">

              <title>Swiss resort</title>
            </head>

            <body>
              <!-- cursor start -->

                <div class="cursor__wrapper">
                  <div class="cursor__follower js-follower"></div>
                  <div class="cursor__label js-label"></div>
                  <div class="cursor__icon js-icon"></div>
                </div>
              </div>
              <!-- cursor end -->

              <main>
                ...
              </main>

              <!-- JavaScript -->

Horizontal and vertical spacing

Control horizontal and vertical spacing of an element using the mx-{size} utilities.

            
              <div class="mx-20"></div>
              <div class="py-20"></div>

Responsive

To control the margin of an element at a specific breakpoint, add a {screen}: prefix to any existing margin utility. For example, adding the class md:mt-20 to an element would apply the mt-20 utility at medium screen sizes and below.

            
              <div class="mt-20 md:mt-15 sm:mt-10"></div>

Global Layout Classes

Classes that are used throughout the template to position page sections and section contents.

            
              .layout-pt-md  { padding-top: 100px; }
              .layout-pt-lg  { padding-top: 200px; }
              
              .layout-pb-md  { padding-bottom: 100px; }
              .layout-pb-lg  { padding-bottom: 200px; }

Lazy Loading

All images (including backgrounds) are loaded in a deferred way to speed up pages loading.

Images

For images in <img> tag, use data-src attribute with .js-lazy class.

            
              <img class="js-lazy" src="#" data-src="lazy.jpg" alt="A lazy image">

Background images

For background images, use data-bg attribute with .js-lazy class.

            
              <div class="bg-image js-lazy" data-bg="lazy.jpg"></div>

Reveal Animations

Single Animation

Template comes with several build in section reveal animations. To add or change an element’s reveal animation, you need to change [data-anim] attribute. 

            
              <div data-anim="slide-up"></div>
              <div data-anim="slide-down"></div>
              <div data-anim="slide-left"></div>
              <div data-anim="slide-right"></div>
              <div data-anim="fade"></div>
              <div data-anim="img-right"></div>

If animation doesn't work properly try to wrap the element you want to animate.

            
              <div data-anim="slide-up">
                Element You want to animate
              </div>

Animation Container

If you want to animate multiple elemets at once, wrap them in data-anim-wrap container and use data-anim-child insted of data-anim attribute. 

            
              <div data-anim-wrap>
                <div data-anim-child="slide-up"></div>
                <div data-anim-child="slide-up"></div>
                <div data-anim-child="slide-up"></div>
              </div>

Delay

By adding additional delay- property, you can change animation delay. Every next value adds 100 milliseconds. 

            
              <div data-anim="slide-up delay-1"></div>
              <div data-anim="slide-up delay-2"></div>
              <div data-anim="slide-up delay-3"></div>
              ...
              <div data-anim="slide-up delay-20"></div>

              <div data-anim-child="slide-up delay-1"></div>
              <div data-anim-child="slide-up delay-2"></div>
              <div data-anim-child="slide-up delay-3"></div>
              ...
              <div data-anim-child="slide-up delay-20"></div>

Parallax Effects

Use [data-parallax] attribute with value on parallax container and [data-parallax-target] to specify parallax item. 

          
            <div data-parallax="0.7" class="h-lg">
              <div
                data-parallax-target
                class="bg-image js-lazy"
                data-bg="some.jpg"
              ></div>
            </div>
